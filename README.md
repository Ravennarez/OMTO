## OMTO Budget Dashboard (Municipality of Bulu Ban)

Frontend-only React dashboard for the Office of the Municipal Tourism Officer (OMTO), showing fiscal year budget allocation and obligation tracking with mock data.

### Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: React Icons
- **State Management**: React Context API

### Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the printed local URL in your browser to view the dashboard.

### Features

- Header with OMTO branding and fiscal year tracker text.
- Transaction panel with **New Transaction** button (mock modal) and quick input (Enter to submit).
- Budget overview cards for each category with progress bars and balances.
- Obligation vs Allocation bar chart and Budget Allocation donut chart using static mock data.
- Responsive transactions table with sample rows and a client-side search field.
- Fully responsive layout for mobile, tablet, and desktop.
